# Flame Analyzer

**Flame Analyzer** — это приложение на PyQt5 для анализа видеопотоков или потоков с камеры с целью измерения и обработки экспериментальных данных (косвенное измерение скорости горения по углу касательной к пламени, напряжение термо-ЭДС) с последующим анализом погрешностей и формированием отчётов.

## Возможности

- Обработка видео/камеры в реальном времени с помощью OpenCV и PyQt5.
- Маскирование и пороговая фильтрация для выделения объектов.
- Автоматический расчёт углов и напряжений по видеокадрам.
- Классификация видео с помощью сверточной нейронной сети. 
- Анализ погрешностей: среднее арифметическое, абсолютная и относительная погрешности, полная погрешность, доверительный интервал.
- Сохранение результатов измерений в CSV или Excel, генерация подробных отчётов.


## Требования

- Python 3.7+
- PyQt5
- OpenCV (`opencv-python`)
- NumPy
- pandas

Установка зависимостей:
```sh
pip install -r requirements.txt
```

Сохранение среды conda:
```sh
conda pack -n .conda -o env.tar.gz
```

## Использование

1. Запустите приложение:
    ```sh
    python main.py
    ```
2. Выберите видеофайл или источник камеры.
3. Настройте чувствительность маски и область интереса.
4. Начните просмотр или анализ видео.
5. Сохраните результаты и отчёты после анализа (папка `output`).

## Структура файлов

- `main.py` - скрипт для запуска приложения.  
- `model.py` - поток для чтения видеопотока и логика эксперимента.  
- `controller.py` — основной код GUI приложения.  
- `view.py` — описание интерфейса PyQt5 (сгенерировано в Qt Designer).  
- `convnet_0.onnx` — сверточная нейросеть для классификации на 30 классов.

## Выходные данные

- Данные измерений сохраняются в `.csv` формате.
- Анализ погрешностей и статистика — в отдельном `.txt` отчёте.
